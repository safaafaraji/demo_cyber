# Cyber Range Backend

Professional backend architecture for Cyber Range platform.

## Features
- **Clean Architecture**: Separation of concerns.
- **REST API**: Fully documented API.
- **Security**: Hardened with Helmet, Rate Limiting, Input Validation.
- **Docker Integration**: Manages labs via Docker API.
- **Monitoring**: Health checks and structured logging.

## Installation

```bash
npm install
npm run dev
```
